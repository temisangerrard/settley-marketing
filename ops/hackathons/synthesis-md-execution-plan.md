## 🎯 Project: Arena Play Autoplay + Filecoin Integration

**Theme Fit:** "Agents that pay, trust, cooperate, keep secrets"

### What We're Building

**1. Autoplay Bot (Core Feature)**
- Users configure bot personality (Aggressive/Conservative/Balanced)
- Bot plays autonomously (RPS, Dice, Heads/Tails)
- Configurable stakes, frequency, stop-loss/take-profit
- Bot persists when user logs off

**2. Admin Dashboard (Judging Feature)**
- Monitor all active bots in real-time
- Filter by personality, game type, stake size
- Aggregate statistics (win rates, profitability)
- Export data for agent evaluation
- A/B test bot configurations

**3. Celo Integration (Bounty Fit)**
- USDC settlements on Celo blockchain
- Transparent, on-chain game outcomes
- Agent wallet management

**4. Filecoin Integration (Autobett — Bring Forward)**
- Store game state on Filecoin (immutable game history)
- Register Arena Play agents using ERC-8004 standard
- Demonstrate verifiable, provably fair gameplay
- Enable decentralized storage for agent training data
