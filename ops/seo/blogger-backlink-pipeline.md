# Blogger Dofollow Backlink Pipeline

**Created:** 2026-03-15  
**Source:** Twitter tip from @kalashvasaniya (600+ likes, 116K impressions)  
**Effort:** ~10 minutes per blog  
**Value:** Free dofollow backlink from high-authority Google domain (blogspot.com)

---

## What Is This?

Blogger.com (owned by Google) provides free dofollow backlinks when you:
1. Create a blog
2. Add your website link in the description or posts
3. Publish content

This is a legitimate, low-effort SEO tactic that compounds over time.

---

## Pipeline Process

### Phase 1: Account Setup (one-time)
- [ ] Create Google account for brand (or use existing)
- [ ] Complete OAuth verification if needed
- [ ] Store credentials securely

### Phase 2: Blog Creation (per blog)
- [ ] Navigate to blogger.com
- [ ] Sign in with brand account
- [ ] Click "Create your blog"
- [ ] Enter blog name (e.g., "Settley - Tokenized Real Estate UK")
- [ ] Choose URL (e.g., settley-uk-tokenized-property.blogspot.com)
- [ ] Set display name (e.g., "Settley")
- [ ] Complete setup

### Phase 3: Content & Links (per blog)
- [ ] Add description with dofollow link to main site
- [ ] Create 1-2 posts with contextual links
- [ ] Publish posts
- [ ] Verify links are live

### Phase 4: Documentation
- [ ] Record blog URL
- [ ] Record post URLs
- [ ] Add to backlink tracker
- [ ] Note date created

## Content Templates

### Blog Description Template
```
[Company] provides [product/service] for [target market]. [Key benefit/value prop]. Visit [URL] to [CTA].
```

**Example:**
> Settley provides tokenized real estate investment for UK and European investors. Fractional property ownership from £100. Visit https://settley.co to learn more about our platform.

### Post Content Template
```
Title: [Keyword-rich title for target audience]

[Hook/question addressing pain point]

[Solution/benefit statement]

[CTA with link]: Visit [URL] to [specific action].

[Optional: additional context or benefits]
```

**Example:**
> Title: Tokenized Real Estate Investment for UK Investors
>
> Looking for fractional property investment in the UK? Settley offers tokenized real estate starting from just £100. Visit https://settley.co to join our waitlist and learn how you can own a share of UK properties with rental income and potential capital appreciation.

## Scaling Strategy

### Target: 10 Blogs in 30 Days

**Week 1:** 3 blogs
- Different niches/angles (e.g., "UK Property Investment", "Fractional Real Estate", "Tokenized Assets")
- Vary content slightly to avoid duplicate content issues

**Week 2-4:** 7 more blogs
- Expand to related topics
- Consider different Google accounts if rate limiting occurs

## Risk Mitigation

**Avoid:**
- Identical content across all blogs (Google may devalue)
- Over-optimization (vary anchor text)
- All blogs created same day (space out)
- Only linking to homepage (mix in deep links if possible)

**Best Practices:**
- Vary blog names and themes
- Write unique descriptions for each
- Space out creation (2-3 per week)
- Use different Google accounts if scaling beyond 10

## Agent Handoff Protocol

This task is ideal for Ralph loop agent execution:

**Task:** `blogger-seo-backlinks`
**Loop Size:** 1 blog per loop (~10 min)
**Completion Criteria:** 10 blogs created in 30 days

**Each Loop:**
1. Create Blogger account (or use existing)
2. Set up blog with unique name/URL
3. Add description with link
4. Create and publish 1 post
5. Log results in tracker
6. Checkpoint and continue

**Checkpoint Format:**

