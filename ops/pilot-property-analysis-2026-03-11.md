## Regulatory Framework (FCA Compliance)

### Whitelist Sale Structure ✅ VIABLE

Based on FCA CP25/28 (October 2025) and current tokenisation guidance:

**Key Requirements:**
1. **Private placement only** - no public marketing of tokens
2. **Appropriateness assessment** - investors must acknowledge high-risk nature
3. **10% net investable assets cap** - retail investors cannot allocate >10% of portfolio
4. **Whitelist architecture** - tokens only transferable to verified eligible addresses
5. **No public prospectus required** - exempt as private placement to limited investors

**Founder's Pass Advantage:**
- Existing members already vetted (10 paying members, 67% conversion)
- Can frame as "pilot investor program" not public token sale
- Tokens represent SPV equity, not regulated security (legal opinion needed)

**Legal Counsel Needed:**
- Confirm SPV structure (UK Limited Company holding property)
- Draft investor terms (equity rights, dividend distribution, exit mechanism)
- Review token smart contract compliance
- **Estimated cost:** £2,000-4,000 for initial setup
