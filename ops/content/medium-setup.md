## 🔧 Technical Setup (Option B: Custom Domain)

**Only do this after 10+ articles published**

### Prerequisites
- Medium publication created
- Custom domain purchased (blog.settley.co or insights.settley.co)
- DNS access to Settley domain

### Steps

**1. Add Custom Domain in Medium**
1. Go to publication settings: https://medium.com/me/publications
2. Click your publication → Settings → Domain
3. Enter: blog.settley.co (or your chosen subdomain)
4. Medium will provide DNS records

**2. Configure DNS Records**

Medium requires:
- **CNAME record** pointing to `medium-customdomains.com`

**Example DNS configuration:**
```
Type: CNAME
Name: blog (or insights)
Value: medium-customdomains.com
TTL: Auto
```

**3. Verify & Activate**
1. Wait 5-30 minutes for DNS propagation
2. Click "Verify" in Medium settings
3. Enable SSL (Medium provides automatically)
4. Set as default domain

**4. Update Existing Posts**
- Medium automatically migrates URLs
- Old medium.com/@settley/... URLs redirect to blog.settley.co/...
- Takes 2-4 weeks for Google to fully update

## 📊 SEO Best Practices for Medium

### On-Article SEO
- **Title:** Include primary keyword in first 60 characters
- **Subtitle:** Add secondary keywords + value prop
- **First paragraph:** Include target keyword naturally
- **Headers:** Use H2/H3 with keyword variations
- **Images:** Add alt text with keywords
- **Tags:** Use all 5 tags (Medium allows 5 per post)

**Example Tags for Settley:**
1. Tokenized Real Estate
2. PropTech
3. Real Estate Investing
4. Fractional Ownership
5. UK Property

### Off-Page SEO
- **Internal linking:** Link to your other Medium articles
- **External linking:** Link to authoritative sources (regulatory docs, market reports)
- **Cross-posting:** Share on Twitter, LinkedIn, Reddit (r/RealEstate, r/PropTech)
- **Newsletter submission:** Submit to Morning Brew, Money Stuff, proptech newsletters

### Medium-Specific SEO Tactics
1. **Publish at optimal times:** Tuesday-Thursday, 8-10 AM EST
2. **Use Medium's distribution:** Submit to large publications (The Startup, Better Marketing, etc.)
3. **Engage with comments:** Reply to every comment in first 24 hours
4. **Curate for Medium Boost:** Well-formatted, original content gets boosted by Medium algorithm
