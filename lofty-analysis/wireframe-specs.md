# Settley Wireframe Specifications (vs Lofty.ai)

## Core Differentiators
1. **UK/EU Focus**  
   - GBP/EUR currency support  
   - Localized property types (terraces vs US condos)  
   - VAT/tax language  

2. **Infrastructure Approach**  
   - "For property owners" path as prominent as "for investors"  
   - API/developer docs linked in footer  

3. **Educational Focus**  
   - "How it works" section before properties  
   - Risk disclosure built into flow  

## Key Screens Needed:
1. **Landing Page**  
   - Dual CTA (Invest vs Tokenize)  
   - Market comparison chart (US vs UK returns)  

2. **Property Listings**  
   - Filter by:  
     - Country (UK primary)  
     - Yield range  
     - Property type  
   - "Why UK properties?" explainer  

3. **Investment Flow**  
   - Step 1: Amount (with GBP/USD toggle)  
   - Step 2: Wallet connection (MetaMask + Coinbase)  
   - Step 3: Legal docs (SPV agreement)  

4. **Owner Dashboard**  
   - Equity release calculator  
   - Tokenization timeline (2-4 weeks)  
   - Management console  

## Dev Notes:
- Prioritize mobile-first (Lofty's mobile experience lacks filters)  
- Build with React components for:  
  - Currency switcher  
  - Yield calculator  
  - Document signer
